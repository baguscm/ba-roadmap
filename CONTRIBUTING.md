# Contributing to BA Roadmap

First off, thanks for taking the time to contribute!

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template when creating an issue.
- Provide as much detail as possible.

### Suggesting Enhancements
- Use the **Feature Request** template.
- Explain the "why" behind your suggestion.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure `npm run lint` and `npm run build` pass before submitting.
3. Link the PR to a related issue if possible.
4. Follow the PR template.

## Style Guide

- **Components**: Use functional components with TypeScript.
- **Styling**: Use Tailwind CSS utility classes.
- **Content**: Ensure BA terminology aligns with APQC or BABOK frameworks.

## Local Development

1. Clone the repo.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000) with your browser.
