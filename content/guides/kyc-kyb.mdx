## Node Knowledge: The Process Lifecycle
1. **Data Collection:** Capturing government-issued IDs (passports, licenses) and liveness biometrics.
2. **Document Authentication:** Validating security features on IDs via automated OCR and AI-base liveness checks.
3. **Database Screening:** Checking customer data against PEP (Politically Exposed Persons) and global sanctions lists.
4. **Adverse Media Check:** Periodic or real-time searching for news records indicating financial crime risk.
