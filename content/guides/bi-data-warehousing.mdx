# BI & Data Warehousing

Falls under "Manage Knowledge and Information" (APQC 12.0) for the e-commerce data stack.

## Node Knowledge: The Process Lifecycle
1. **Extraction (ETL):** Pulling data from the ERP, WMS, CRM, and Website logs.
2. **Transformation:** Cleaning and normalizing dates, names, and cross-channel currencies.
3. **Loading:** Moving data into a centralized Star or Snowflake schema for analysis.
4. **Querying:** Using SQL or BI tools to build complex multi-dimensional models.

## Key BA Activities (BABOK® Alignment)

In alignment with the *Business Analysis Body of Knowledge (BABOK®)*, mastering this node relies on the following knowledge areas and techniques:

*   **Data Modelling:** Designing Dimensions (Date, SKU, Region) and Measures (Revenue, AOV).
*   **Business Capability Analysis:** Understanding what strategic decisions the data warehouse must support.
*   **Risk Analysis:** High-priority management of PII (Personally Identifiable Information).
